Leap.DeviceList¶

class Leap.DeviceList¶

The DeviceList class represents a list of Device objects.

Get a DeviceList object by calling Controller.devices().

New in version 1.0.

classmethod DeviceList()¶

Constructs an empty list of devices.

New in version 1.0.

len -> int

The number of items in this list.

New in version 1.0.

is_empty -> bool

Reports whether the list is empty.

New in version 1.0.

append(other)¶

Appends the members of the specifed DeviceList to this DeviceList.

Arguments

other (Leap.DeviceList) – A DeviceList object containing Device objects to append to the end of this DeviceList.

New in version 1.0.
