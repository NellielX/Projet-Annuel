Leap.GestureList¶

class Leap.GestureList¶

The GestureList class represents a list of Gesture objects.

Get a GestureList object by calling Frame.gestures().

New in version 1.0.

classmethod GestureList()¶

Constructs an empty list.

New in version 1.0.

len -> int

The number of items in this list.

New in version 1.0.

is_empty -> bool

Reports whether the list is empty.

New in version 1.0.

append(other)¶

Appends the members of the specifed GestureList to this GestureList.

Arguments

other (Leap.GestureList) – A GestureList object containing Gesture objects to append to the end of this GestureList.

New in version 1.0.
